# nmpc_robo

## Descrição
Este projeto implementa um Controle Preditivo Baseado em Modelo (NMPC) para robôs, oferecendo um modelo matematico para o tal e controle do mesmo para o seguimento de uma linha
identificada via OpenCV.

## Tecnologias
- Python
- ROS
- C
